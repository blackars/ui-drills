# Playable Interface Lab

A Micro Arcade Laboratory — minigames, prototypes, and interactive surfaces exploring the web as a playable medium.

## Live Site

**[blackars.github.io/PlayableInterfacesLaboratory](https://blackars.github.io/PlayableInterfaceLaboratory/)**

## What Is This

A living archive of 26 interactive HTML/CSS/JavaScript experiments. Zero frameworks. Zero build steps. Pure vanilla web.

The project uses minimal visual systems and serial artistic production as core tools for radical conceptual experimentation with UI, game design, and generative art.

## Tag System

Each module is tagged for filtering:

| Tag | Description |
|-----|-------------|
| `game` | Playable minigames with score/loop/mechanics |
| `experience` | Interactive visual prototypes and demos |
| `element` | Reusable web components for projects |
| `test` | Quick experiments and proof of concepts |

## Tech

- **HTML5** + **CSS3** + **JavaScript** (vanilla)
- No frameworks, no build tools, no dependencies
- Canvas API, CSS transforms, IntersectionObserver
- Dark monochrome aesthetic, minimal borders, clean typography
